# Коррекции

Если пользователь поправляет агента — записать ошибку и правильный вариант, чтобы не повторять.

---

## Шаблон коррекции

```
### [Дата] — Коррекция

**Что агент сделал неправильно:** ...

**Как правильно:** ...

**Почему:** ...

**Контекст:** (клиент / ситуация)
```
