# Кейсы клиентов

Формат записи кейса — после каждой консультации или значимого разбора.

---

## Шаблон кейса

```
### [Дата] — [ФИО / инициалы] — [Возраст, пол]

**Главный запрос:** ...

**Ключевые показатели:**
| Показатель | Значение | Статус |
|---|---|---|
| ... | ... | 🔴 / 🟡 / 🟢 |

**Диагнозы / состояния:** ...

**Назначения:**
- Нутрицевтики: ...
- Питание: ...
- Образ жизни: ...

**Результат (при наличии динамики):**
- Через ___ мес.: показатель X вырос с ___ до ___
- Самочувствие: ...

**Выводы / что запомнить:**
- ...
```
