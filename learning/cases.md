```

**Ключевые показатели:**
| Показатель | Значение | Статус |
|---|---|---|
| ... | ... | 🔴 / 🟡 / 🟢 |

**Диагнозы / состояния:** ...

**Назначения:**
- Нутрицевтики: ...
- Питание: ...
- Образ жизни: ...

**Результат (при наличии динамики):**
- Через ___ мес.: показатель X вырос с ___ до ___
- Самочувствие: ...

**Выводы / что запомнить:**
- ...
```

**Диагнозы / состояния:** Аденомиоз (N80.0), обильные менструации (N92.0), ЖДА хроническая (D50.0), нетоксический одноузловой зоб 1 ст. (E04.1), подтверждённая ИР, скрытый гипотиреоз, курение.
