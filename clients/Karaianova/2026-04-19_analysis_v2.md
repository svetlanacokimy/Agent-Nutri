Ферритин — внутриклеточный белок-депо, состоящий из 24 субъединиц (тяжёлые Н-цепи + лёгкие L-цепи). Одна молекула ферритина способна хранить до 4500 атомов Fe³⁺. Сывороточный ферритин отражает размер общего депо железа в РЭС (печень, селезёнка, костный мозг).

### 2.4. Углеводный обмен — биохимическая глубина

| Показатель | Результат | Ед. | Реф. лаб. | Оптимум | Статус |

| **Инсулин** | **13,6** | мкЕд/мл | 2–29,1 | 5–7 | 🔴 |

**Формула:** HOMA-IR = Инсулин × Глюкоза ÷ 22,5 = 13,6 × 5,63 ÷ 22,5 = **3,40**

#### Биохимический механизм ИР у данного клиента

В норме: инсулин → связывание с инсулиновым рецептором (IR) → аутофосфорилирование тирозиновых остатков → активация субстрата инсулинового рецептора 1 (IRS-1) → фосфоинозитид-3-киназа (PI3K) → протеинкиназа В (Akt/PKB) → транслокация GLUT4 из внутриклеточных везикул на мембрану → захват глюкозы.

При ИР: на каком-то этапе этого каскада — блок. Вероятные причины у данного клиента:
1. **Курение** → ↑ кортизол → серинфосфорилирование IRS-1 (вместо тирозинового) → ↓ передача сигнала
2. **Дефицит витамина D** (52,3 < 60) → ↓ экспрессия гена инсулинового рецептора (витамин-D-рецептор (VDR) — транскрипционный фактор гена IR)
3. **Вероятный дефицит магния (Mg²⁺)** (не сдан!) → Mg²⁺ — кофактор тирозинкиназы IR; при дефиците Mg → ↓ аутофосфорилирование IR → ↓ активация IRS-1 → ИР. Дефицит Mg увеличивает риск ИР в 6–7 раз (refs/04)

**С-пептид 3,68** (оптимум ≤2,5) — С-пептид секретируется β-клетками ПЖ эквимолярно инсулину (проинсулин → инсулин + С-пептид). Повышение подтверждает, что гиперинсулинемия — эндогенная (не экзогенная), ПЖ работает в режиме компенсаторной гиперсекреции.

> ⚠️ **TOFI-феномен (Thin Outside, Fat Inside):** ИМТ 19,63 при HOMA-IR 3,40. ИР возможна при любом весе. Механизм: висцеральная жировая ткань (даже в малом количестве) + хронический стресс (курение) + дефициты (D, Mg) → нарушение инсулинового каскада.

### 2.5. Щитовидная железа — биохимическая глубина

| Показатель | Результат | Ед. | Реф. лаб. | Оптимум | Статус |
|---|---|---|---|---|---|
| **ТТГ** | 0,779 | мЕд/л | 0,4–4,0 | 0,8–2,0 | 🟡 |
| **Т3 свободный** | 2,45 | пг/мл | 1,58–3,91 | Середина (~2,75) | 🟡 |
| **Т4 свободный** | 10,31 | пмоль/л | 9,01–19,05 | Выше середины (~14) | 🟡 |
| **АТ-ТПО** | 2,15 | МЕ/мл | 0–5,61 | Идеально 0 | 🟢 |

#### Биохимический механизм нарушения функции ЩЖ

**Синтез Т4 (тироксин):**
В норме: ТТГ → рецептор ТТГ на тироците → активация натрий-йодового симпортера (NIS) → захват I⁻ → ТПО окисляет I⁻ → I⁰ → йодирование тирозиновых остатков тиреоглобулина → монойодтирозин (МИТ) + дийодтирозин (ДИТ) → конденсация МИТ + ДИТ = Т3, ДИТ + ДИТ = Т4 → протеолиз тиреоглобулина → секреция Т3/Т4 в кровь.

**Проблема №1 — ↓ синтез Т4:**
ТПО — гем-содержащий фермент (простетическая группа = гем). При ферритине 7,98 → ↓ доступность гема → ↓ активность ТПО → ↓ йодирование тиреоглобулина → ↓ Т4. Результат: Т4 10,31 (при середине референса 14,03 — на 26% ниже середины).

**Проблема №2 — ↓ конверсия Т4→Т3:**
80% Т3 образуется периферически (печень, почки, мышцы) путём дейодирования Т4. Фермент — ДИ2 (дейодиназа 2 типа) — это селеноцистеин-содержащий фермент, ЗАВИСИМЫЙ от Fe (Fe входит в состав ферредоксина, который является электронным донором для ДИ2). При ↓ Fe → ↓ ДИ2 → ↓ конверсия Т4→Т3. Результат: Т3 2,45 (при середине 2,75 — на 11% ниже).

**Проблема №3 — нетипичный паттерн ТТГ:**
ТТГ 0,779 — чуть ниже оптимума (0,8–2,0). По классической отрицательной обратной связи гипоталамо-гипофизарно-тиреоидной оси (ГГТО), при ↓ Т4/Т3 ТТГ должен расти. Возможные объяснения:
1. **Частичная автономия узла** (1,6 × 1,35 × 2,23 см, TIRADS 2) — узел может производить небольшое количество Т4 неконтролируемо → подавление ТТГ тиреотрофами гипофиза
2. **↓ Fe → ↓ тиреотрофы** — тиреотрофы гипофиза тоже Fe-зависимы в синтезе/секреции ТТГ
3. **Курение → ↑ кортизол → подавление тиреолиберина (ТРГ)** — кортизол ингибирует паравентрикулярное ядро гипоталамуса → ↓ ТРГ → ↓ стимуляция тиреотрофов → ↓ ТТГ

**Вывод:** скрытый гипотиреоз — «узел + нормальные гормоны = скрытый гипотиреоз» (refs/06). Гормоны формально в референсе, но функционально недостаточны.

**Данные УЗИ ЩЖ (18.03.2026):**
- Правая доля: 1,77 × 1,61 × 4,75 см → объём 7,25 мл
- Левая доля: 1,26 × 1,23 × 4,42 см → объём 3,57 мл
- Общий объём: 10,82 мл (норма для женщин ≤18 мл)
- Узел правой доли: 1,6 × 1,35 × 2,23 см, TIRADS 2, Bethesda II (доброкачественный коллоидный узел с кистозной дегенерацией)
- Кальцитонин 4,18 пг/мл (норма <5) → злокачественность исключена

## 3. РАБОТА С ЗАБОЛЕВАНИЯМИ

### 3.1. ЖДА — корневая проблема №1

**Этиопатогенез у данного клиента:**
```
Аденомиоз (N80.0) — эктопия эндометрия в миометрий
  ↓
Менометроррагия (N92.0) — обильные менструации со сгустками
  ↓
Хроническая потеря Fe: менструация = ~30 мл крови = ~15 мг Fe
(при норме потери = 1–2 мг/день, суточное всасывание = 1–2 мг)
  ↓
Дисбаланс: потери >> всасывание → последовательное истощение:
1) ферритин (депо) ✅ → 7,98 (истощён)
2) сывороточное Fe (транспорт) — не сдано, предположительно ↓
3) трансферрин ↑ компенсаторно — не сдан
4) гемоглобин ✅ → 101 (↓)
  ↓
↓ синтез гема (феррохелатаза) → ↓ Hb → ↓ кислородная ёмкость крови
→ тканевая гипоксия → ↑ ЭПО → ↑ эритропоэз (ретикулоциты 1,23% — норма,
но ограничен Fe-субстратом) → гипохромия (MCH 27,2↓)
```

**Стратегия коррекции:**
1. **Устранение причины потери:** ВМС-ЛНГ (Мирена) — ↓ менструальную кровопотерю на 70–90%
2. **Восполнение Fe:** бисглицинат железа — см. подробное обоснование в рекомендациях
3. **Кофакторы:** витамин С (↑ абсорбцию Fe²⁺ через редукцию Fe³⁺→Fe²⁺ + восстановление ферропортина)

### 3.2. Инсулинорезистентность — корневая проблема №2

**Каскад нарушений:**
```
↓ чувствительность IR/IRS-1/PI3K/Akt
  ↓
Компенсаторная гиперинсулинемия (инсулин 13,6)
  ↓
├─→ ↑ SREBP-1c → ↑ липогенез → ТГ 1,33↑, ЛПНП 2,7↑
├─→ ↑ ароматаза (CYP19A1) → ↑ эстрон → эстрогендоминирование → аденомиоз?
├─→ ↑ IGF-1 → ↑ mTOR → пролиферация
├─→ С-пептид 3,68 → ПЖ в режиме гиперсекреции → риск истощения β-клеток
└─→ Глюкоза 5,63 → стадия компенсации → без коррекции → преддиабет → СД 2 типа
```

**Стратегия:** берберин (активатор аденозинмонофосфат-активируемой протеинкиназы (AMPK)) + Mg²⁺ (кофактор тирозинкиназы IR) + витамин D (экспрессия гена IR) + питание (↓ гликемическая нагрузка) + физическая активность (↑ GLUT4 через AMPK-независимый путь)

### 3.3. Субоптимальная функция ЩЖ — вторична к Fe-дефициту

**Порочный круг:**
```
↓ Fe (ферритин 7,98) → ↓ гем-содержащие ферменты:
├─→ ↓ ТПО → ↓ йодирование тиреоглобулина → ↓ Т4 (10,31)
└─→ ↓ ДИ2 (+ ↓ Se?) → ↓ конверсия Т4→Т3 → ↓ Т3 (2,45)
       ↓
  ↓ метаболизм → ↓ моторика ЖКТ → ↓ кислотность → ↓ абсорбция Fe²⁺
       ↓
  ↓ Fe ещё → 🔄 ПОРОЧНЫЙ КРУГ
```

**Точка разрыва:** агрессивное восполнение Fe → ↑ ТПО → ↑ Т4; ↑ ДИ2 → ↑ Т3 → ↑ метаболизм → ↑ абсорбция Fe → положительная спираль.

### 3.4. Курение — системный агравирующий фактор

Биохимические механизмы влияния курения на все выявленные проблемы:
- **ЖДА:** ↓ аскорбат (вит. С) → ↓ редукция Fe³⁺→Fe²⁺ в просвете кишечника → ↓ абсорбция через двухвалентный металлотранспортёр 1 (DMT1)
- **ИР:** никотин → ↑ кортизол → серинфосфорилирование IRS-1 → ↓ PI3K/Akt → ↓ GLUT4
- **ЩЖ:** тиоцианаты (SCN⁻, продукт метаболизма цианида из табачного дыма) → конкурентное ингибирование NIS → ↓ захват йода тироцитами
- **Липиды:** ↑ свободные радикалы → ↑ ПОЛ → окЛПНП → атерогенез
- **Гомоцистеин:** курение → истощение В12, В9, В6 → ↑ гомоцистеин → повреждение эндотелия
