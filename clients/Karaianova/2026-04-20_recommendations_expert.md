| **Механизм** | Fe²⁺ через PepT1 → энтероцит → ферропортин (FPN1) → плазма → гефестин + церулоплазмин окисляют Fe²⁺→Fe³⁺ → трансферрин → костный мозг (феррохелатаза → гем → Hb), печень (CYP450), ЩЖ (ТПО, ДИ2), митохондрии (Fe-S кластеры) |
| **Корректирует** | Ферритин 7,98 → 60–90; Hb 101 → ≥120; MCH 27,2 → ≥28; Т4/Т3 ↑ (через ТПО/ДИ2) |
| **Ожидаемый эффект** | 7–10 дней: ретикулоцитарный криз; 4 нед.: Hb +10–15 г/л; 8 нед.: Hb ≥120, ферритин ≥25–35; 3–6 мес.: ферритин 60+ |

## 5. РЕКОМЕНДАЦИИ ПО ОБРАЗУ ЖИЗНИ

### 5.1. Курение — НАСТОЯТЕЛЬНО РЕКОМЕНДУЕТСЯ БРОСИТЬ

- Каждая сигарета → −25 мг вит. С + ↑ расход Se, Zn, вит. Е, GSH
- Тиоцианаты (SCN⁻) → ↓ NIS → ↓ захват йода ЩЖ → при узловом зобе — противопоказано
- Никотин → ↑ кортизол → ↑ глюконеогенез + серинфосфорилирование IRS-1 → ↑ ИР
- Кадмий → ↓ реабсорбция Fe в почках

**Если бросить сразу невозможно:** ↑ вит. С до 1000 мг; постепенно ↓ количество сигарет.

## 7. МИРЕНА (ВМС-ЛНГ)

Рекомендована гинекологом при аденомиозе.

**Позитивный эффект:** ↓ менструальной кровопотери на 70–90% → ускоренное восстановление ферритина.

**Нутрициологический аспект:** левоноргестрел обладает андрогенной активностью → может ↑ ИР.

**Рекомендация:** мониторинг HOMA-IR через 3 мес. после установки; при ↑ → усилить протокол ИР.
