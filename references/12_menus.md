| Меню | Для кого | Особенности |

## Безмолочное безглютеновое меню (файл «Традиционное безмолочное и безглютеновое меню»)

**Назначение:** быстрая нормализация работы кишечника (результат через 3–5 дней).

### Ключевые правила
- При запорах 2–3 дня → растительное слабительное (сенаде 2 табл.) перед началом
- Пробиотики 1–2 мес. (отменить после нормализации стула)
- Подключить желчегонный курс
- Плотные белки → обед; лёгкие → завтрак/ужин
- Питаться по голоду (оценка 4 балла)
- Можно на постоянной основе — сбалансировано по БЖУ и нутриентам
- Если стул стал жидким → убрать 1 порцию салата + прекратить желчегонные

### Тест на переносимость
Когда кишечник ежедневно опорожняется 2 недели → съесть 1 блюдо с глютеном/молоком → наблюдать 2 дня. Затруднения → продукт не подходит.

### Структура дня
| Приём | Пример |
|-------|--------|
| Завтрак | 2 яйца + зелёный салат + киноа (белая/красная/чёрная) |
| Перекус | Фрукты или ягоды |
| Обед | Индейка/курица + гречка/рис + томатный соус + свежий салат |
| Перекус | Чай + орехи одного вида |
| Ужин (до 19:00) | Стейк рыбы + запечённые овощи |

## Лакто-вегетарианское меню (файл «Лакто-вегетарианское меню»)

**2-недельное меню.** Можно адаптировать: добавить мясо/рыбу или убрать молочные.

### 30 рекомендаций (ключевые)

**Сочетаемость:**
- Не смешивать сырые овощи и фрукты → газообразование
- Не есть бобовые + картофельное пюре → вздутие
- Не есть сладкое после бобовых; сладкое → через 2,5–3 ч после белкового обеда
- Перед белковой пищей → тарелка сырого салата

**Белок:**
- Качественный белок = бобово-злаковые блюда (кичри и т.п.) 3–4 р/нед. в обед
- Метионин + лизин = сочетание бобовых + злаков

**Кальций:**
- Кунжут 2–3 ст.л. 2–3 р/нед. (рекордное количество кальция): белый жареный или чёрный замоченный

**Питьё:**
- Запрещено запивать еду холодной водой. Можно — тёплой (30–40°) маленькими глотками
- 1–2 стакана воды после сна; до 1 л — в первой половине дня

**Режим:**
- Сон до 22:00, подъём 6–7:00
- Сладкое = отдельный приём пищи, 1–2 р/нед.
- Не несколько блюд одновременно («первое-второе-компот» → забыть)
- 1 приём в день — не слишком белковый (завтрак или ужин)

## Аутоиммунный протокол (AIP) (файл «Аутоиммунное меню»)

**Палео аутоиммунный протокол** — питание + образ жизни для управления АИЗ.

### Порядок перехода
1. Начать с безмолочного/безглютенового меню (2–3 мес.)
2. После адаптации → перейти на AIP
3. После стабилизации → возвращать продукты по 1 каждые 2 дня через дневник питания

### ЧТО ЕСТЬ (AIP)
- Мясо: курица, индейка, кролик
- Субпродукты: печень, сердце, почки, язык (исключить при ↑ холестерине)
- Рыба и морепродукты (дикая предпочтительнее)
- Все овощи (вся радуга цветов), зелень
- Водоросли (кроме хлореллы/спирулины — иммуностимуляторы!)
- Грибы
- Травы и специи
- Полезные жиры: животные, жирная рыба, оливковое/авокадо/кокосовое масло
- Фрукты: 1–3 в день, не больше
- Ферментированные продукты: квашеные овощи, кокосовый кефир/йогурт, пробиотики

### ЧТО ИЗБЕГАТЬ (AIP)
- Зерновые (все)
- Бобовые (все)
- Молочные и кисломолочные (включая сыр, мороженое)
- Рафинированный сахар, подсластители (даже стевия), мёд, кленовый сироп
- Рафинированные масла
- Яйца
- Орехи (все — из-за ω-6 → воспаление)

### Дополнительно при AIP
- Пробиотики обязательно
- Омега-3 обязательно
- Сон 21–22:00
- При запорах — свекольно-морковный салат 1–2 р/день
- Дисбактериоз + дырявый кишечник = главные триггеры АИЗ

## Меню на 1300 ккал (файл «Меню на 1300 ккал»)

Если можно глютен и молочные → добавить. Если расчёт > 1300 ккал → добавить крупы/орехи.

### Примерный день

| Приём | Пример | КБЖУ |
|-------|--------|------|
| Завтрак | Сл. масло 10 г + пармезан 40 г + 4 хлебца + чай + зефир в шоколаде 1 шт. | 380/15/19/35 |
| Обед | Индейка 100 г + гречка 150 г + сл. масло | 387/34/15/28 |
| Перекус | Апельсин | 62/1/0/15 |
| Ужин | Семга 150 г + цвет. капуста 150 г | 343/32/20/9 |
| **ИТОГО** | | **~1300/85/56/89** |

**Вино:** 150 г белого сухого = 112 ккал = пустые калории. Учитывать в КБЖУ!

## Меню при запорах и диарее (файл «Меню таблицы при запорах и диарее»)

### При ЗАПОРАХ
Акцент на: сырые салаты (½ тарелки × 2 р/день), свёклу, киноа (красная/чёрная — богаты клетчаткой), зелёные салаты, квашеную капусту.

| Завтрак | Обед | Ужин |
|---------|------|------|
| 2 яйца + салат + киноа / фритата / поке / панкейки / каша | Индейка/курица/нут/чечевичный суп + салат | Рыба + запечённые овощи / фунчоза / рисовая лапша |

### При ДИАРЕЕ
Акцент на: термообработанные овощи, каши (овсяная, льняная), пропаренный рис, банан, хлебцы. Меньше сырого.

| Завтрак | Обед | Ужин |
|---------|------|------|
| Овсяная каша / льняная каша / шакшука / яйца + свекольный салат | Рыба/курица + запечённые овощи / бобовый суп + хлебцы | Овощная подлива / фритата / фунчоза / рыба + киноа |

## Рекомендации для диабетиков и ИР (файл «Рекомендации для диабетиков»)

### Ключевые принципы
1. ↑ белок + полезные жиры к каждому приёму углеводов (жиры замедляют всасывание глюкозы)
2. Исключить перекусы → **3 плотных приёма пищи в день**
3. Между ужином и завтраком: **10–12 часов**
4. Плотный завтрак с белком/жирами стабилизирует сахар на весь день
5. Сон до 22:30–23:00 (поздний → ↑ кортизол → ИР)
6. Использовать глюкометр для контроля реакции на продукты

### 9 продуктов с высоким ГИ — ограничить/исключить

| Продукт | ГИ (сырой) | ГИ (приготовленный) |
|---------|-----------|-------------------|
| Кабачок | — | 75 |
| Картофель | — | 65–96 (зависит от способа) |
| Морковь | 40 | 85 |
| Свёкла | 30 | 65 |
| Белый рис | — | 89–90 (басмати/бурый = 50) |
| Пшено | — | 71 |
| Перловка | — | 70 |
| Тыква | — | 75 |
| Ананас | — | 65 |

### Полезные жиры (использовать)
Сливочное/топлёное масло, яичные желтки, кокосовое масло, оливковое масло

### Вредные масла (исключить)
Кукурузное, соевое, подсолнечное, маргарин (промышленная обработка → воспаление)

### Исключить
1. Сладкие напитки, фруктовые соки, газировки, подслащённый кофе/чай
2. Конфеты, промышленная/домашняя сладкая выпечка
3. Мёд (макс. 1 ч.л./день), сладкие сиропы
4. Промышленная молочная продукция (перейти на домашнюю или исключить)

### Образец рациона при ИР / диабете II типа

| Приём | Блюда |
|-------|-------|
| 1-й завтрак | Фритата (1 яйцо + 5–6 белков + зелёная фасоль + перец) + киноа + сырые овощи/квашеная капуста |
| Обед | Суп из нута/чечевицы + хлебцы с печенью трески **ИЛИ** 2–3 котлеты из птицы + гречка + свежий салат |
| Ужин | Тушёные/запечённые овощи (стручковая фасоль, цвет. капуста, брокколи, перец) + рыба |
| Перекус (→ убрать) | 7–8 замоченных миндаля + травяной чай **ИЛИ** 100 г ягод **ИЛИ** 1 фрукт с низким ГИ |

**Лучшие углеводы при ИР:** бобовые, корнеплоды, ягоды.

## Универсальные правила питания (общие для всех меню)

1. **Голод** — основной показатель приёма пищи (оценка 4 балла)
2. **Не переедать**, но наедаться в основные приёмы
3. **Сырой салат перед белком** — помогает перевариванию
4. **Бобовые замачивать на ночь**; после них — не есть сладкое (через 2,5–3 ч)
5. **Вода:** 1–2 стакана после сна, до 1 л в первой половине дня, не запивать еду холодной водой
6. **Сон:** до 22:00, подъём 6–7:00 → «ложимся СЕГОДНЯ, встаём ЗАВТРА»
7. **Не смешивать:** сырые овощи + фрукты; бобовые + картофельное пюре
8. **Сладкое** = отдельный приём пищи, 1–2 р/нед., в идеале → заменить фруктами
9. **Не несколько блюд:** забыть «первое-второе-компот»
10. **Кунжут** 2–3 ст.л. 2–3 р/нед. = лучший источник кальция
