## Этап 2. Оценка КАЖДОГО показателя (ни один не пропускается!)

Для каждой строки таблицы анализов выполняй строго по порядку:

### Алгоритм «ячейка за ячейкой»

```
1. Ячейка «Показатель» → идентифицируй параметр
2. Ячейка «Результат» → числовое значение клиента
   ⚠️ Если пусто → статус = НЕ СДАН → добавить в список «досдать»
   ⚠️ Не интерпретируй пустую ячейку как референс или другие данные!
3. Ячейка «Референс» → лабораторные границы нормы
4. Ячейка «Оптимальные значения» → из references/11_reference_values.md
5. Оценка → присвой статус
```

### Таблица статусов

| Статус | Условие | Действие |
|--------|---------|----------|
| 🔴 ПАТОЛОГИЧЕСКОЕ ОТКЛОНЕНИЕ | Результат вне референса | Немедленно фиксируется, требует коррекции |
| 🟡 СУБОПТИМАЛЬНОЕ ЗНАЧЕНИЕ | В референсе, но вне оптимума | Фиксируется как субклиническое отклонение |
| 🟢 В ПРЕДЕЛАХ ОПТИМУМА | Результат в оптимуме | Проверить в контексте совокупной картины |
| ⚪ НЕ СДАН | Результат отсутствует | В список «досдать» с обоснованием |

### Правило контекстного анализа

Даже если показатель 🟢 В ПРЕДЕЛАХ ОПТИМУМА — анализируй в контексте:
- Есть связанные показатели с отклонениями?
- Есть симптомы клиента, указывающие на скрытую проблему?
- Пример: гемоглобин 130 (оптимум), но ферритин 15 → латентная анемия

## Этап 3. Клинические связки (проверяй ВСЕГДА!)

После оценки всех показателей — обязательно проверь каждую профильную связку:

| Связка | Какие показатели проверить | Что искать |
|--------|---------------------------|------------|
| **Анемия** | Гемоглобин + ферритин + железо + В12 + В9 + трансферрин + MCV + MCH + MCHC | Тип анемии (ЖДА / В12-дефицитная / фолат-дефицитная / смешанная), латентная стадия |
| **Щитовидная железа** | ТТГ + Т3св + Т4св + АТ-ТПО + АТ-ТГ + йод + селен + цинк + железо | Гипо/гипертиреоз, АИТ, нарушение конверсии Т4→Т3, связь с ферритином/дефицитами |
| **Инсулинорезистентность** | Инсулин + глюкоза + HOMA-IR + гликир. гемоглобин + С-пептид + триглицериды | ИР, преддиабет, метаболический синдром. HOMA-IR = (инсулин × глюкоза) / 22,5 |
| **Печень / желчь** | Билирубин (общ/прям/непрям) + АЛТ + АСТ + коэф. де Ритиса (АСТ/АЛТ) + ГГТ + ЩФ | Холестаз, жировой гепатоз, синдром Жильбера, нарушение желчеоттока |
| **Воспаление** | СРБ + лейкоциты + нейтрофилы + СОЭ + лимфоциты | Хроническое вялотекущее воспаление, острый процесс, аутоиммунная активность |
| **Паразитоз** | Эозинофилы + лимфоциты↑ + нейтрофилы↓ + антитела к паразитам | Паразитарная инвазия, необходимость АПК |
| **Белковый обмен** | Общий белок + альбумин + мочевина + креатинин | Белковый дефицит, нарушение усвоения, мальабсорбция |
| **Липидный профиль** | Холестерин + ЛПВП + ЛПНП + триглицериды + коэф. атерогенности + гомоцистеин | Атерогенный профиль, кардиориски, гипергомоцистеинемия |

### Примеры перекрёстных связей

- Низкий ферритин + низкий Т3 → нарушена конверсия Т4→Т3 (железо — кофактор дейодиназы)
- Высокий гомоцистеин + низкие В12/В9/В6 → дефицит метилирования
- Высокие лимфоциты % + низкие нейтрофилы % → подозрение на паразитоз или вирусную нагрузку
- Высокий инсулин + нормальная глюкоза → ранняя ИР (глюкоза «держится» за счёт гиперинсулинемии)
- Низкий общий белок + низкая мочевина → дефицит белка в рационе или мальабсорбция
- АСТ > АЛТ (коэф. де Ритиса > 1,3) → подозрение на сердечную патологию; < 0,9 → печень

## Этап 4. Reasoning Chains — логические причинно-следственные цепочки

После проверки всех строк и связок — построй причинно-следственные цепочки между показателями.

### Формат цепочки

```
Показатель А (значение, статус) → следствие 1 → следствие 2 → связь с Показателем Б → клинический вывод
```

### Виды цепочек

| Тип | Описание | Как обозначать |
|-----|----------|----------------|
| **Линейная** | А → Б → В | `→` |
| **Порочный круг** | А усугубляет Б, Б усугубляет А | `🔄 ПОРОЧНЫЙ КРУГ` |
| **Корневая причина** | Одно отклонение порождает каскад | `🎯 КОРНЕВАЯ ПРИЧИНА` |
| **Компенсация** | Организм компенсирует дефицит за счёт другого ресурса | `⚖️ КОМПЕНСАЦИЯ` |

### Правила построения

1. **Начинай с показателей 🔴** — они чаще всего являются корневыми или критичными
2. **Ищи ПОРОЧНЫЕ КРУГИ** — когда А усугубляет Б, а Б усугубляет А → разорвать цикл = приоритет
3. **Ищи КОРНЕВУЮ ПРИЧИНУ** — одно-два отклонения, от которых тянутся остальные
4. **Проверяй кофакторные связи** — дефицит кофактора → блокирует ферментативную реакцию → вторичное отклонение
5. **Учитывай анамнез** — хроническое заболевание или лекарство может быть корневой причиной

### Примеры цепочек

#### Пример 1: Порочный круг «Железо ↔ Щитовидка»
```
🎯 Ферритин 7.98 (🔴) → истощение запасов Fe
  → нарушена работа дейодиназы (требует Fe как кофактор)
  → снижена конверсия Т4 → Т3
  → функциональный гипотиреоз (Т3↓)
  → гипотиреоз замедляет перистальтику ЖКТ
  → снижается всасывание Fe в кишечнике
  → ферритин падает ещё ниже
  🔄 ПОРОЧНЫЙ КРУГ: Fe ↔ ЩЖ
  💡 Точка разрыва: начать с восполнения железа (бисглицинат)
```

#### Пример 2: Каскад от аденомиоза
```
🎯 Аденомиоз (диагноз)
  → обильные менструации → хроническая потеря Fe
  → ферритин падает → железодефицитная анемия
  → ткани не получают O₂ → усталость, выпадение волос, одышка
  → дейодиназа без Fe → нарушение конверсии Т4→Т3
  → функциональный гипотиреоз → отёки, набор веса, запоры
  💡 Корневая причина — хроническая кровопотеря, решение — железо + работа с гинекологом
```

#### Пример 3: Инсулинорезистентность + воспаление
```
Инсулин 14.2 (🔴) + глюкоза 5.8 (🟡) → HOMA-IR 3.67 (🔴)
  → гиперинсулинемия → стимуляция липогенеза
  → триглицериды ↑ → ЛПНП ↑ → атерогенный профиль
  → хроническое воспаление (СРБ 3.2 🟡)
  → воспаление усиливает ИР
  🔄 ПОРОЧНЫЙ КРУГ: ИР ↔ воспаление
  💡 Точка разрыва: берберин + питание + снижение веса
```

#### Пример 4: Дефицит метилирования
```
Гомоцистеин 18.4 (🔴) → нарушен цикл метилирования
  → В12 280 (🟡) + В9 5.1 (🔴) + В6 12 (🔴) — кофакторы метилирования истощены
  → токсичный гомоцистеин повреждает сосуды → кардиориск
  → нарушено метилирование ДНК → риск онко
  🎯 КОРНЕВАЯ ПРИЧИНА: дефицит В9 + В12 + В6
  💡 Коррекция: метилфолат + метилкобаламин + P-5-P + ТМГ
```

### На каждую цепочку обязательно:
- Указать точку разрыва (с чего начать коррекцию)
- Пометить порочные круги символом 🔄
- Пометить корневые причины символом 🎯

## Этап 5. Автоматические алерты (Alerts)

При обнаружении следующих паттернов — **НЕМЕДЛЕННО** вывести алерт перед основным анализом.

### Блокирующие алерты (🚫) — запрещают действия

| # | Условие | Алерт | Действие |
|---|---------|-------|----------|
| 1 | Ферритин < 20 | 🚫 **АПК ЗАПРЕЩЁН** — ферритин ниже 20, сначала поднять запасы железа | Не назначать АПК |
| 2 | АТ-ТПО повышены + клиент принимает/запрошен йод | 🚫 **Йод > 150 мкг ОПАСЕН при АИТ** — высокие дозы йода усиливают аутоиммунную атаку на ЩЖ | Йод ≤ 150 мкг или отменить |
| 3 | Ферритин < 20 + запоры | 🚫 **Двойной блок для АПК** — ферритин И стул не готовы | Сначала Fe + стул → потом АПК |

## Этап 6. Мета-анализ (Meta-Analysis)

После всех этапов — составь системную мета-сводку.

### 6.1. Затронутые системы (ранжировать по тяжести)

```
| Приоритет | Система / Орган | Статус | Ключевые показатели |
|-----------|----------------|--------|---------------------|
| 1 | ... | 🔴 Критично | ... |
| 2 | ... | 🟡 Субоптимально | ... |
| 3 | ... | 🟢 Мониторинг | ... |
```

### 6.2. Корневая причина

Определи 1–2 ГЛАВНЫХ причины, от которых тянется каскад остальных отклонений:

```
🎯 КОРНЕВАЯ ПРИЧИНА #1: [название]
   Механизм: [как она порождает остальные отклонения]
   Затронутые показатели: [список]
   Приоритет коррекции: ПЕРВЫЙ

🎯 КОРНЕВАЯ ПРИЧИНА #2: [название] (если есть)
```

### 6.3. Прогноз

| Сценарий | 1 месяц | 3 месяца | 6 месяцев |
|----------|---------|----------|-----------|
| **Без коррекции** | Что ухудшится | Что усугубится | Риски |
| **При соблюдении рекомендаций** | Первые улучшения | Ожидаемая динамика | Целевые значения |

#### Правила прогнозирования

- Опирайся на типичные сроки коррекции дефицитов:
  - Ферритин: +10–15 единиц за 1–2 мес. при бисглицинате
  - Витамин D: +10–20 единиц за 2 мес. при 5000 МЕ/день
  - В12: заметный рост через 4–6 нед.
  - Гомоцистеин: снижение через 6–8 нед. на метилированных формах
  - HOMA-IR: улучшение через 2–3 мес. при берберине + диете
  - ТТГ: стабилизация через 2–3 мес. при коррекции кофакторов
- Прогноз «без коррекции» — куда пойдут показатели, какие симптомы усилятся
- Прогноз НЕ заменяет врачебный — это ориентир

## Этап 7. Системный клинический вывод

После проверки всех строк и связок — составь сводку:

1. **Какие органы/системы затронуты** — перечислить с приоритетом (от критичных к субклиническим)
2. **Какие метаболические цепочки нарушены** — связать показатели в цепочки причин
3. **Возможные компенсаторные механизмы** — что организм компенсирует и за счёт чего
4. **Предполагаемые состояния/диагнозы** — на основе анализов + симптомов + связок
5. **Требуемые направления коррекции** — нутрицевтики / питание / образ жизни / досдать анализы
