# Скилл: Анализ крови и лабораторных данных

Перед выполнением — прочитай RULES.md.

## Назначение

Скилл выполняет экспертный разбор анализов и формирует аналитическую основу для рекомендаций.

## Входные данные

1. clients/[ФИО]/blood_tests.md и/или фото анализов
2. clients/[ФИО]/profile.md
3. references/11_reference_values.md
4. Профильные references по выявленным отклонениям

## Последовательность

1. Запустить проверку единиц через skills/unit_conversion.md.
2. Обработать каждый показатель: результат, референс, оптимум, статус.
3. Зафиксировать отклонения и потенциальные риски.
4. Построить причинно-следственные цепочки и порочные круги.
5. Определить корневые причины.
6. Сформировать список «что досдать и зачем».
7. Подготовить системный экспертный вывод.

## Авто-алерты

1. Проверять блокирующие риски до назначения нутрицевтиков.
2. При критических комбинациях выносить алерт в начало разбора.
3. При неполных данных не останавливать работу: формировать вывод на основании имеющихся данных.

## Выход

Файл clients/[ФИО]/[ДАТА]_analysis_expert.md со структурой из RULES.md.

## Передача дальше

Передать ключевые выводы в skills/recommend.md для сборки схемы коррекции.
