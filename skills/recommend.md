# Скилл: Формирование нутрицевтических рекомендаций

Перед выполнением — прочитай RULES.md.

## Назначение

Скилл превращает клинические выводы в пошаговый план коррекции.

## Входные данные

1. clients/[ФИО]/[ДАТА]_consult_expert.md (или аналитический блок текущего /consult до финальной сборки)
2. clients/[ФИО]/profile.md
3. references/09_nutraceuticals.md
4. references/05_vitamins_minerals.md
5. Профильные references по диагнозам клиента
6. text_extracted/Ссылки БАД.txt

## Последовательность

1. Приоритизировать цели коррекции по этапам: критично, важно, поддержка.
2. Для каждого нутрицевтика указать: зачем, форма, дозировка, схема, срок эффекта, ограничения.
3. Проверить совместимость приёма по времени.
4. Выполнить обязательную проверку лекарственных взаимодействий с текущими лекарствами клиента.
5. Сформировать блок питания и КБЖУ под цель клиента.
6. Раздел КБЖУ обязателен и включает:
  - шаг 1: исходные данные (пол, возраст, рост, вес, индекс массы тела, физическая активность);
  - шаг 2: базовый обмен веществ по формуле Миффлина-Сен-Жеора с подстановкой чисел клиента;
  - шаг 3: коэффициент активности (какой выбран и почему);
  - шаг 4: целевая калорийность (поддержание/дефицит/набор) с клиническим обоснованием;
  - шаг 5: распределение БЖУ (белки, жиры, углеводы) в граммах и процентах с обоснованием;
  - шаг 6: распределение по приёмам пищи;
  - шаг 7: норма воды (30 мл × кг).
7. Сформировать календарь контроля и пересдачи анализов.
8. Добавить раздел «Где купить» только по назначенным позициям.

## ПРОВЕРКА ЛЕКАРСТВЕННЫХ ВЗАИМОДЕЙСТВИЙ (обязательный раздел)

Для каждого назначаемого нутрицевтика проверить взаимодействие с лекарствами клиента.

Минимум проверок по умолчанию:
1. Берберин: ингибирует CYP3A4, CYP2D6
2. Омега-3: усиливает антикоагулянты
3. Магний: снижает абсорбцию тетрациклинов
4. Витамин К2: антагонист варфарина

Если лекарства не указаны, обязательно вынести предупреждение:
"Лекарственные взаимодействия не проверены — данные о лекарствах не предоставлены"

## ПОДДЕРЖКА МИКРОБИОМА (после нутрицевтиков, обязательный раздел)

1. При инсулинорезистентности добавить поддержку микробиома:
  - пробиотики: Lactobacillus rhamnosus, Bifidobacterium lactis (в контексте оси кишечник-инкретины и GLP-1)
2. При назначении берберина учитывать его антимикробный эффект и добавлять пробиотическую поддержку по этапам.
3. Добавлять пребиотики: инулин, пектин, резистентный крахмал (с титрацией по переносимости).
4. Добавлять ферментированные продукты (без сахара) по переносимости.
5. Если у клиента выраженные признаки СИБР, ввод клетчатки и пробиотиков делать ступенчато с контролем симптомов.

Источники для блока микробиома:
1. text_extracted/УРОК 4. КИШЕЧНИК. СЛАДКОЕ. ЗАВТРАК.txt
2. text_extracted/УРОК 8. СИБР. СДК. Анализ по Осипову.txt

## Режимы

1. EXPERT: глубокая аргументация и патофизиологическая связка с показателями.
2. CLIENT: тёплая адаптация в понятный язык по структуре RULES.md (8 разделов).
