# Скилл: Golden recommend

Перед выполнением — прочитай RULES.md.

## Назначение

Скилл собирает разделы тактики без дрейфа структуры.
Зона ответственности: только разделы 6-9 итоговой expert-консультации и адаптация в client-версию.
Граница анализа и терапии: см. RULES.md раздел 1.7.
Preflight и режим запуска: см. RULES.md 1.6 и 1.6.1.
Метрики глубины: см. RULES.md 1.10.

## Входные данные

1. Нормализованный аналитический блок (разделы 1-5)
2. clients/[ФИО]/profile.md
3. references/09_nutraceuticals.md
4. references/05_vitamins_minerals.md
5. Профильные references по диагнозам клиента
6. text_extracted/Ссылки БАД.txt
7. Для калибровки глубины: clients/Karaianova/2026-04-19_recommendations_v2.md

## Контракт входа

1. REQUIRED поля входа:
  - режим запуска: STRICT или FALLBACK
  - пол, возраст, рост, вес
  - жалобы
  - лекарства
  - ключевые отклонения анализов
2. Если вход невалиден в режиме STRICT: вернуть stop-report с конкретным списком отсутствующих полей.
3. Если вход невалиден в режиме FALLBACK: продолжать с явной маркировкой ограничений.

## Контракт выхода

1. Выход в expert: один блок разделов 6-9 с фиксированным порядком подпунктов ниже.
2. Выход в client: адаптация по RULES.md 3.2 без [Источник: ...].
3. Без таблиц в итоговых документах.

## Детерминированная структура expert-разделов

### 6.1 КБЖУ

Использовать строго этот шаблон:
1. Исходные данные клиента.
2. Формула Миффлина-Сен-Жеора.
3. Расчёт с подстановкой чисел клиента.
4. Обоснование пропорций БЖУ с привязкой к диагнозам.
5. Распределение по приёмам пищи.
6. Вода по формуле 30 мл × кг.

Мини-шаблон текста:
- Исходные данные: [пол], [возраст], [рост], [вес], [ИМТ], [активность].
- БОВ: [формула] = [число] ккал.
- Коэффициент активности: [коэф], потому что [клиническая логика].
- Цель калорийности: [поддержание/дефицит/набор] = [число] ккал.
- БЖУ: белки [г], жиры [г], углеводы [г], с клинической аргументацией.
- Распределение: завтрак [ккал], обед [ккал], перекус [ккал], ужин [ккал].
- Вода: [вес] × 30 мл = [мл].

### 6.2 TRIAGE

Использовать строго этот шаблон:
1. 🔴 Критично (не более 5 позиций, этап 6-8 недель).
2. 🟡 Важно (этап 6-8 недель после стабилизации критичного блока).
3. 🟢 Поддержка (долгосрочно).

Мини-шаблон текста:
- 🔴 Критично: [позиция 1], [позиция 2] ...
- 🟡 Важно: [позиция 1], [позиция 2] ...
- 🟢 Поддержка: [позиция 1], [позиция 2] ...

### 6.3 НУТРИЦЕВТИКИ (главный раздел)

Для каждого ключевого нутрицевтика использовать этот шаблон:
1. Биохимическое обоснование формы: транспортёр всасывания + сравнение альтернатив.
2. Расчёт дозы: формула с цифрами клиента.
3. Механизм действия: полный каскад живым языком.
4. Корректирует: текущее значение -> целевое значение.
5. Ожидаемый эффект: 2 нед / 4 нед / 8 нед / 3 мес.
6. Взаимодействия: что усиливает, что блокирует, интервалы по часам.

Обязательный эталонный пример (из v2) для формата блока:
"Железа бисглицинат — 80 мг элементарного Fe.
Биохимическое обоснование формы: бисглицинат железа — хелатная форма, всасывается через PepT1 и имеет лучшую переносимость.
Сравнение альтернатив: сульфат/фумарат чаще дают побочные эффекты ЖКТ, бисглицинат переносится мягче.
Расчёт дозы: ферритин 7,98 при цели 60-90, старт 80 мг элементарного Fe в день.
Механизм: восстановление депо через трансферрин и ферритин, усиление эритропоэза, рост гемоглобина.
Корректирует: ферритин 7,98 -> 60+; гемоглобин 101 -> 120+.
Ожидаемый эффект: 2 нед — ретикулоцитарный ответ; 4 нед — рост Hb; 8 нед — рост ферритина; 3 мес — стабилизация депо.
Взаимодействия: не сочетать с чаем/кофе/антацидами, интервал с кальцием не менее 2 часов, синергия с витамином C.
Схема: утром натощак, курс 3-6 месяцев с контролем анализов."

### 7.1 ДИЕТИЧЕСКИЕ РЕКОМЕНДАЦИИ

Шаблон для исключения:
1. Продукт/группа к исключению.
2. Механизм вреда живым языком.
3. Привязка к показателям клиента.

Шаблон для добавления:
1. Продукт/группа к добавлению.
2. Ключевые нутриенты.
3. Механизм пользы.
4. Какой клинический дефицит/риск закрывает.

Обязательный пример из v2:
"Исключить быстрые углеводы при HOMA-IR 3,40, потому что они усиливают гиперинсулинемию и липогенез.
Добавить печень говяжью 2-3 раза в неделю как источник гемового железа, B12, фолата и меди для коррекции железодефицитной анемии."

### 7.2 ПИТЬЕВОЙ РЕЖИМ

Использовать строго этот шаблон:
1. Когда пить.
2. Сколько пить.
3. Чем дополнять.
4. Что исключить и почему.

Мини-шаблон текста:
- Утро после пробуждения: [объём].
- До еды: [объём и интервал].
- В течение дня: [объём].
- Дополнить: [травяные чаи/варианты], если переносимость позволяет.
- Исключить: [сладкие напитки/алкоголь/избыточный кофе] с клиническим объяснением.

### 7.3 ПРИМЕРНОЕ МЕНЮ НА 1 ДЕНЬ

Использовать строго этот шаблон:
1. Завтрак + КБЖУ.
2. Перекус + КБЖУ.
3. Обед + КБЖУ.
4. Перекус + КБЖУ.
5. Ужин + КБЖУ.

Мини-шаблон текста:

- Перекус 1: [блюда]. КБЖУ: [числа].
- Обед: [блюда]. КБЖУ: [числа].
- Перекус 2: [блюда]. КБЖУ: [числа].
- Ужин: [блюда]. КБЖУ: [числа].
- Итого за день: [ккал], белки [г], жиры [г], углеводы [г].

### 7.4 ОБРАЗ ЖИЗНИ

Шаблон для вредных привычек:
1. Курение/алкоголь: биохимия вреда живым языком.
2. Практическая стратегия снижения/отказа.

Шаблон для сна:
1. Длительность и окно сна.
2. Привязка к мелатонину, инсулину, щитовидной железе.

Шаблон для спорта:
1. Нагрузка с привязкой к GLUT4.
2. Ограничения при анемии и критерии безопасного прогресса.

### 7.5 ГИНЕКОЛОГИЧЕСКИЙ/ГОРМОНАЛЬНЫЙ СТАТУС (если женщина)

Использовать строго этот шаблон:
1. Аденомиоз/эндометриоз -> кровопотеря -> железо.
2. КОК/ВМС -> влияние на инсулинорезистентность.
3. Менопауза -> витамин D, кальций, магний.
4. Связка с патологическими цепочками из разделов 1-5.

### 8 КОНТРОЛЬНЫЕ ТОЧКИ

2. 6 недель.

На каждой точке обязательно:

3. Решения по корректировке схемы.

### 9 КЛИНИЧЕСКИЙ ВЫВОД

Всегда 2 сценария прогноза:

## Обязательные блоки (всегда присутствуют)

1. Проверка лекарственных взаимодействий.
2. Поддержка микробиома (триггер: инсулинорезистентность или берберин).
3. Где купить (только позиции из назначений, ссылки из text_extracted/Ссылки БАД.txt).

### Проверка лекарственных взаимодействий

Минимальные проверки:
1. Берберин: CYP3A4/CYP2D6.
2. Омега-3: антикоагулянты.
3. Магний: тетрациклины/фторхинолоны.
4. Витамин K2: варфарин.

Если лекарства не указаны, выводить:
"Лекарственные взаимодействия не проверены — данные о лекарствах не предоставлены".

### Поддержка микробиома

Обязательные элементы при активном триггере:
1. Пробиотики: Lactobacillus rhamnosus, Bifidobacterium lactis.
2. Пребиотики: инулин, пектин, резистентный крахмал, титрация по переносимости.
3. Ферментированные продукты без сахара.
4. Ступенчатый ввод при симптомах СИБР.

## Режимы

### Expert-режим

1. Источники обязательны по RULES.md 1.9.
2. Глубина блоков по RULES.md 1.10.
3. Полный механизм и клиническая привязка к цифрам клиента.

### Client-режим

1. Источники запрещены по RULES.md 1.9.
2. Смысл сохраняется, язык упрощается без потери причинно-следственной логики.
3. Тон: врач-практик по RULES.md 2.2.

## Anti-recursion

1. Если вход уже содержит финальный consult_expert, использовать аналитическую часть только как read-only контекст.
2. Не повторять генерацию тактики поверх уже сгенерированных блоков без сравнения динамики.
