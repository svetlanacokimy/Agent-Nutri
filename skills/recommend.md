## Алгоритм работы

### Шаг 0. Triage — приоритизация назначений

**ПЕРЕД** подбором нутрицевтиков — классифицируй ВСЕ выявленные отклонения по приоритету:

| Приоритет | Маркер | Критерий | Примеры |
|-----------|--------|----------|---------|
| **КРИТИЧНО** | 🔴 | Патологические отклонения, требующие немедленной коррекции | Ферритин < 20, гемоглобин < 110, HOMA-IR > 2.7, витамин D < 20, В12 < 300, гомоцистеин > 15 |
| **ВАЖНО** | 🟡 | Значимые субоптимальные значения, ухудшающие качество жизни | Ферритин 20–40, вит. D 20–40, ТТГ > 2.5, инсулин 8–10, В9 < 10, магний < 0.8 |
| **ВНИМАНИЕ** | 🟢 | Мониторинг, профилактика, оптимизация | Показатели близко к оптимуму, но не в нём; фоновая поддержка |

### Правило этапности

Назначения разбиваются на ЭТАПЫ. На каждом этапе — **не более 5–7 нутрицевтиков одновременно**.

| Этап | Срок | Что назначаем | Цель |
|------|------|---------------|------|
| **Этап 1** | 4–6 недель | Только 🔴 КРИТИЧНО | Устранить опасные дефициты, разорвать порочные круги |
| **Этап 2** | 4–6 недель | 🟡 ВАЖНО + продолжение 🔴 при необходимости | Коррекция субклинических отклонений |
| **Этап 3** | Поддержка | 🟢 ВНИМАНИЕ + профилактика | Оптимизация, долгосрочная поддержка |

### Правила Triage

1. **Сначала корневые причины** — если из Reasoning Chains выявлена корневая причина → её коррекция идёт в Этап 1
2. **Порочные круги → разорвать на Этапе 1** — определи точку разрыва и начни с неё
3. **Не более 5–7 одновременно** — если 🔴-назначений больше → приоритизируй по клинической значимости
4. **Контроль между этапами** — перед переходом к Этапу 2 оценить динамику (самочувствие, при возможности — повторный анализ)
5. **АПК — только Этап 2 или 3** — после нормализации ферритина, стула, желчеоттока

### Формат вывода Triage

```
## Triage — приоритеты коррекции

### 🔴 КРИТИЧНО (Этап 1 — немедленно)
1. Ферритин 7.98 → бисглицинат железа 50 мг
2. Витамин D 18 → холекальциферол 10 000 МЕ
3. ...

### 🟡 ВАЖНО (Этап 2 — через 4–6 недель)
1. Магний 0.72 → цитрат магния 400 мг
2. ...

### 🟢 ВНИМАНИЕ (Этап 3 — поддержка)
1. Омега-3 → профилактика 1 г EPA+DHA
2. ...
```

### Шаг 1. Определение оснований для назначения

## Обоснование каждого назначения (ОБЯЗАТЕЛЬНО)

Для КАЖДОГО нутрицевтика обязательно указывать все 6 пунктов:

1. **Название и форма** — с обоснованием выбора формы на уровне биохимии (какой транспортёр, биодоступность, побочные эффекты)
2. **Дозировка** — с обоснованием: почему именно такая доза, ссылка на references/
3. **Механизм действия** — какой биохимический/метаболический путь восстанавливает (ферменты, рецепторы, сигнальные каскады)
4. **Какой показатель корректирует** — конкретное число из анализов клиента
5. **Ожидаемый эффект** — через сколько недель и что изменится (прогноз по показателям)
6. **Взаимодействия** — с чем нельзя совмещать, с чем усиливается эффект

### Шаблон обоснования

```
### [Нутрицевтик] — [Форма]

**Биохимическое обоснование формы:** [почему именно эта форма — транспортёр, биодоступность, сравнение с другими формами]

**Дозировка:** [доза] — [обоснование: степень дефицита, вес клиента, ссылка на references/]

**Механизм действия:** [какой путь восстанавливает: фермент → субстрат → продукт → эффект]

**Корректирует:** [показатель] = [текущее значение] → целевое [значение] (оптимум из references/11)

**Ожидаемый эффект:** через [N] нед./мес.: [что изменится в показателях и самочувствии]

**Взаимодействия:** 
- 🚫 Не совмещать с: [список]
- ✅ Усиливается при приёме с: [список]

**[Источник: references/XX_filename.md, строки NN-MM]** ← ОБЯЗАТЕЛЬНО
```

### Примеры обоснований

**Железа бисглицинат:**
- **Форма:** бисглицинат — хелатная форма, где Fe²⁺ связано с двумя молекулами аминокислоты глицина. Всасывается через пептидный транспортёр PepT1 в тонком кишечнике (а не через DMT1, как неорганическое железо — сульфат, фумарат). Преимущества: а) не конкурирует с Ca²⁺ и Zn²⁺ за DMT1; б) не генерирует свободные радикалы (реакция Фентона) в просвете кишечника; в) биодоступность в 2–4 раза выше сульфата; г) минимальные побочные эффекты со стороны ЖКТ (нет запоров, тошноты). Источник: references/05_vitamins_minerals.md

**Берберин:**
- **Механизм:** активирует АМФ-активируемую протеинкиназу (AMPK) — «метаболический переключатель» клетки → ↑ транслокация GLUT4 на мембрану → ↑ захват глюкозы миоцитами → ↓ глюкоза крови; параллельно ↓ SREBP-1c → ↓ липогенез в печени → ↓ ТГ; ↓ экспрессия PCSK9 → ↑ рецепторы ЛПНП → ↓ ЛПНП. Источник: references/04_lipids_insulin.md

### Формат для продуктов к добавлению
```
**Добавить: [продукт/группа]**
Ключевой нутриент: [что содержит] → закрывает дефицит: [какой] → биохимический эффект: [описание]

```

### Примеры

**Исключить: быстрые углеводы (белый хлеб, выпечка, сладости)**
→ Механизм: высокий гликемический индекс (ГИ) → резкий ↑ глюкозы → ↑ выброс инсулина → при существующей ИР (HOMA-IR 3,40) нагрузка на β-клетки поджелудочной → истощение → прогрессия в преддиабет

**Добавить: печень говяжья (2–3 р/нед)**
→ Содержит: гемовое железо (6,9 мг/100г, биодоступность 15–35% vs 2–10% у негемового), витамин В12 (60 мкг/100г), фолат, медь → закрывает дефицит: Fe-депо (ферритин 7,98), потенциально В12 → эффект: гемовое Fe всасывается через HCP1 (Heme Carrier Protein 1), не конкурирует с фитатами и таннинами, быстрее поднимает ферритин чем растительные источники

## Расчёт КБЖУ (ОБЯЗАТЕЛЬНЫЙ раздел)

В каждой консультации расчёт КБЖУ выполняется ОБЯЗАТЕЛЬНО по следующему алгоритму:

### Шаг 1. Базовый обмен (формула Миффлина-Сан Жеора)

**Женщины:** БОВ = Вес × 9,99 + Рост × 6,25 − Возраст × 4,92 − 161
**Мужчины:** БОВ = Вес × 9,99 + Рост × 6,25 − Возраст × 4,92 + 5

### Шаг 2. Коэффициент активности (из references/02_nutrition_basics.md)

| Уровень активности | Коэффициент |
|---|---|
| Минимальная (офис, без тренировок) | ×1,2 |
| 2 тренировки/нед | ×1,4 |
| 4–5 тренировок/нед | ×1,46 |
| 5–6 тренировок/нед | ×1,55 |
| Ежедневные тренировки | ×1,63 |
| 2 тренировки/день | ×1,72 |

### Шаг 3. Целевая калорийность

- **Похудение:** норма − 10–20% (минимум 1200–1300 ккал)
- **Поддержание:** норма
- **Набор:** норма + 10–20%

### Шаг 4. Распределение БЖУ (г/кг массы тела)

Из references/02_nutrition_basics.md:
- **Белки:** 1,2–1,6 г/кг (сидячий → умеренно активный), до 1,8–2 г/кг при спорте 5 р/нед
- **Жиры:** 1 г/кг (не уменьшать!)
- **Углеводы:** 1,6–2 г/кг (без спорта) — до 3 г/кг (↑ спорт)

### Шаг 5. Распределение по приёмам пищи

| Приём | % от суточной калорийности |
|---|---|
| Завтрак | 25–30% |
| Обед | 35–40% |
| Ужин | 20–25% |
| Перекусы (если есть) | 10–15% |

### Формат вывода

```
## Расчёт КБЖУ

**Исходные данные:** пол [Ж/М], вес [кг], рост [см], возраст [лет], активность [описание]

**БОВ:** [вес] × 9,99 + [рост] × 6,25 − [возраст] × 4,92 − 161 = [ккал]
**С учётом активности:** [БОВ] × [коэф.] = [ккал]
**Цель:** [похудение/поддержание/набор] → [итоговые ккал]

**БЖУ:**
| Макронутриент | г/кг | Граммы | Ккал |
|---|---|---|---|
| Белки | [X] г/кг | [граммы] | [× 4] |
| Жиры | [X] г/кг | [граммы] | [× 9] |
| Углеводы | — | [остаток] | [× 4] |
| **ИТОГО** | | | [сумма] |
```

### Шаг 2. Подбор лучшей формы

Для КАЖДОГО нутрицевтика указывай наиболее усваиваемую форму:

| Нутрицевтик | Лучшие формы | НЕ рекомендуемые формы |
|---|---|---|
| Железо | Бисглицинат (хелат) | Сульфат железа (плохо переносится ЖКТ) |
| Магний | Цитрат, глицинат, таурат | Оксид (низкая биодоступность) |
| Витамин D | D3 (холекальциферол) | D2 (менее эффективен) |
| Витамин С | Эстер-С, аскорбат натрия, липосомальный | — |
| В12 | Метилкобаламин | Цианокобаламин (хуже усваивается) |
| В9 | Метилфолат (5-МТГФ), метилтетрагидрофолат | Фолиевая кислота (синтетическая, не конвертируется у части людей) |
| В6 | Пиридоксал-5-фосфат (P-5-P) | Пиридоксин гидрохлорид |
| Йод | Йодид калия, в смеси йода | Раствор Люголя без контроля |
| Селен | Селенометионин | Селенит натрия (менее безопасен) |
| Цинк | Пиколинат, бисглицинат | Оксид |
| Омега-3 | Триглицеридная форма, EPA+DHA | Этиловые эфиры |
| Кальций | Цитрат | Карбонат (при низкой кислотности не усвоится) |
| Медь | Хелат (бисглицинат) | — |

### Шаг 3. Определение дозировки

- Используй дозировки из `references/05_vitamins_minerals.md` и `references/09_nutraceuticals.md`
- Учитывай степень дефицита: чем глубже — тем выше стартовая доза
- Учитывай вес клиента (примеры: магний 15–20 мг × массу тела, вит. D по уровню в крови)
- Учитывай противопоказания: камни в желчном → магний не более 400–600 мг
- При АИЗ → проверить ограничения из `references/09_nutraceuticals.md` (таблица АИЗ-ограничений)

### Шаг 4. Формирование схемы распределения по времени суток

#### Правила совместимости (СТРОГО!)

| Правило | Описание |
|---------|----------|
| Железо ОТДЕЛЬНО | НЕ с кальцием, НЕ с цинком, НЕ с магнием — разносить по времени |
| Железо + вит. С | Вместе — С повышает усвоение железа |
| Кальций ОТДЕЛЬНО | НЕ с железом, НЕ с магнием |
| Йод ОТДЕЛЬНО | НЕ с железом и кальцием в один приём |
| Магний вечером | Способствует усвоению йода и витаминов, расслабляет, улучшает сон |
| Витамин D с жирами | Принимать во время еды с жирами (обед) |
| В12 + В9 вместе | Можно совмещать |
| Пробиотик отдельно | Через 2 часа после еды или перед сном |
| Сорбенты отдельно | За 1,5–2 часа до или после всего остального |
| Берберин до еды | За 15–30 мин до приёма пищи |
| Омега-3 с жирами | Во время еды, разделить на 2 приёма (обед + ужин) |

## Правила для АПК (антипаразитарный курс)

Перед назначением АПК ОБЯЗАТЕЛЬНО проверь 6 условий:

| # | Условие | Если НЕ выполнено |
|---|---------|-------------------|
| 1 | Ферритин ≥ 20 | Сначала поднять ферритин |
| 2 | Нет запоров | Сначала наладить стул |
| 3 | Желчеотток работает | Сначала наладить желчеотток |
| 4 | НЕ беременна / НЕ кормит | АПК запрещён |
| 5 | Ребёнок ≥ 4 лет | АПК запрещён до 4 лет |
| 6 | Камни в ЖП | < 1 см → НЕЛЬЗЯ АПК; > 1 см → можно |

Если все условия выполнены → загрузи `references/10_antiparasitic.md` для выбора схемы (Аврора / универсальная).

**Пример:** высокий гомоцистеин → Этап 1: В12 + В9 + В6 + ТМГ → после снижения гомоцистеина → Этап 2: добавить железо + медь.
