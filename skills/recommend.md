# Скилл: Формирование нутрицевтических рекомендаций

Иерархия источников правды: см. `v2/HIERARCHY.md`.
Процесс мышления (пайплайн `/consult`): см. `v2/PIPELINES.md`.
Перед выполнением — прочитай RULES.md.

## Назначение

Скилл превращает клинические выводы в пошаговый план коррекции.

## Последовательность

### Граница с аналитическими разделами (обязательно)

1. Этот скилл заполняет только тактические блоки (разделы 6-9 итоговой expert-консультации).
2. Не переносить нутрицевтики в разделы 1-5 (анализ, цепочки, корневые причины, досдать).

Правило разделения факта и терапии: см. `RULES.md §1.7`.

### ПРАВИЛО ЖИВОГО ЯЗЫКА

1. Объяснять механизмы как живой клинический процесс, а не как конспект по биохимии.
2. Запрещены сухие цепочки без контекста и объяснений типа "активация X -> Y".
3. Каждый механизм связывать с цифрами клиента и практическим смыслом для его состояния.
4. Любую аббревиатуру расшифровывать в контексте первого упоминания.

1. Приоритизировать цели коррекции по этапам: критично, важно, поддержка.
2. Для каждого нутрицевтика указать: зачем, форма, дозировка, схема, срок эффекта, ограничения.
2.1. В режиме EXPERT для каждого ключевого нутрицевтика обязательны 6 смысловых слотов (по `v2/STYLE_GUIDE.md`):
  - **Слот 1 — Почему именно эта форма:** биохимическое обоснование транспортёра/всасывания + сравнение минимум с одной альтернативой;
  - **Слот 2 — Расчёт дозы от цифр пациента:** формула с подставленными числами этого пациента;
  - **Слот 3 — Что происходит в клетке:** полный механизм живым языком (ферменты, рецепторы, транспортёры в клиническом контексте);
  - **Слот 4 — Корректирует:** текущее значение пациента → целевое значение;
  - **Слот 5 — Ожидаемая динамика:** по точкам 2 нед / 4 нед / 8 нед / 3 мес;
  - **Слот 6 — Взаимодействия и приём:** синергии, антагонисты, время приёма, интервалы.

  Если какой-то слот пуст — блок недописан. Формальный счётчик строк не применяется.
3. Проверить совместимость приёма по времени.
4. Выполнить обязательную проверку лекарственных взаимодействий с текущими лекарствами клиента.
5. Сформировать блок питания и КБЖУ под цель клиента.
5.1. Для каждого ключевого продукта к исключению/добавлению в режиме EXPERT давать минимум 8 строк объяснения с привязкой к диагнозам и анализам клиента.
6. Раздел КБЖУ обязателен и включает:
  - шаг 1: исходные данные (пол, возраст, рост, вес, индекс массы тела, физическая активность);
  - шаг 2: базовый обмен веществ по формуле Миффлина-Сен-Жеора с подстановкой чисел клиента;
  - шаг 3: коэффициент активности (какой выбран и почему);
  - шаг 4: целевая калорийность (поддержание/дефицит/набор) с клиническим обоснованием;
  - шаг 5: распределение БЖУ (белки, жиры, углеводы) в граммах и процентах с обоснованием;
  - шаг 6: распределение по приёмам пищи;
  - шаг 7: норма воды (30 мл × кг).
7. Сформировать календарь контроля и пересдачи анализов.
7.1. В разделе "Клинический вывод и риски" фиксировать прогноз в двух сценариях (без коррекции/при соблюдении) на 1/3/6/12 месяцев.
8. Добавить раздел «Где купить» только по назначенным позициям.

## Автопроверка глубины перед финализацией (обязательно)

Перед сохранением итогового expert-документа выполнить self-check по 5 уровням:
1. Нутрицевтики: каждый ключевой блок содержит все 6 обязательных слотов (форма/доза-расчёт/механизм-в-клетке/целевое-значение/динамика-2-4-8-12нед/взаимодействия). Пустой слот = недописанный блок.
2. Причинно-следственные цепочки: минимум 7 звеньев и биохимические узлы на каждом шаге.
3. Диетические решения: минимум 8 строк на ключевой продукт с привязкой к диагнозам и показателям.
4. КБЖУ: полная формула Миффлина-Сен-Жеора с подстановкой чисел, обоснование пропорций по клиническому сценарию.
5. Клинический вывод: единая системная модель и прогноз 1/3/6/12 месяцев в двух сценариях.

## ПРОВЕРКА ЛЕКАРСТВЕННЫХ ВЗАИМОДЕЙСТВИЙ (обязательный раздел)

Для каждого назначаемого нутрицевтика проверить взаимодействие с лекарствами клиента.

Минимум проверок по умолчанию:
1. Берберин: ингибирует CYP3A4, CYP2D6
2. Омега-3: усиливает антикоагулянты
3. Магний: снижает абсорбцию тетрациклинов
4. Витамин К2: антагонист варфарина

Если лекарства не указаны, обязательно вынести предупреждение:
"Лекарственные взаимодействия не проверены — данные о лекарствах не предоставлены"

## ПОДДЕРЖКА МИКРОБИОМА (после нутрицевтиков, обязательный раздел)

1. При инсулинорезистентности добавить поддержку микробиома:
  - пробиотики: Lactobacillus rhamnosus, Bifidobacterium lactis (в контексте оси кишечник-инкретины и GLP-1)
2. При назначении берберина учитывать его антимикробный эффект и добавлять пробиотическую поддержку по этапам.
3. Добавлять пребиотики: инулин, пектин, резистентный крахмал (с титрацией по переносимости).
4. Добавлять ферментированные продукты (без сахара) по переносимости.
5. Если у клиента выраженные признаки СИБР, ввод клетчатки и пробиотиков делать ступенчато с контролем симптомов.

Источники для блока микробиома:
1. text_extracted/УРОК 4. КИШЕЧНИК. СЛАДКОЕ. ЗАВТРАК.txt
2. text_extracted/УРОК 8. СИБР. СДК. Анализ по Осипову.txt
