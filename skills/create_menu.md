## 8 шагов составления меню

### ШАГ 1. Анализ здоровья клиента (база перед меню)

На основе данных клиента определи:

**Источники данных:**
- Клинические и биохимические анализы крови
- Копрограмма, анализ мочи
- КЩБ (кислотно-щелочной баланс)
- Симптомы, образ жизни, жалобы

**Что учесть:**
- Диагнозы: ИР, СИБР, гипотиреоз, анемия, воспаления ЖКТ, слабая ферментативная активность, аутоиммунные процессы, аллергии, эндометриоз, мастопатия
- Индивидуальные особенности: состояние ЖКТ, желчеотток, КЩБ, режим сна, физическая активность, пищевые привычки, тип телосложения (экто/мезо/эндоморф)

**Результат шага 1:** чёткий список ограничений и нутриентных приоритетов

Примеры:
- «Исключить глютен и молочку, повысить белок до 1,5 г/кг»
- «Исключить продукты AIP-протокола (паслёновые, яйца, орехи, семена)»
- «Низкоуглеводное питание, 3 приёма без перекусов, 10–12 ч ночной перерыв»

### ШАГ 2. Сбор пожеланий по продуктам и предпочтениям

Спросить у клиента или взять из анкеты:
- Какие продукты любит / не любит и никогда не будет есть
- Уровень кулинарной подготовки (сложные рецепты или «ленивые» решения?)
- Стиль жизни (работает из дома / в офисе / разъезды)
- Бюджет (важно для подбора рецептов)
- Нужны ли meal-prep решения (приготовить заранее / заморозить)

### ШАГ 3. Расчёт КБЖУ и объёмов порций

**Формулы — из `references/02_nutrition_basics.md`**

1. Рассчитать базовый обмен (BMR)
2. Умножить на коэффициент активности → TDEE
3. В зависимости от цели:
   - Похудение: TDEE − 15–20%
   - Поддержание: TDEE
   - Набор массы: TDEE + 10–15%

4. Распределить по макронутриентам:
   - Белки: 1,2–2,0 г/кг (зависит от цели и диагнозов)
   - Жиры: 0,8–1,2 г/кг
   - Углеводы: остаток калоража

5. Распределить по приёмам пищи (% от суточной нормы зависит от типа тела)

### ШАГ 4. Формирование структуры питания

| Тип телосложения | Количество приёмов | Особенности |
|---|---|---|
| Эктоморф | 5 (3 основных + 2 перекуса) | Чаще, меньшими порциями, повышенный калораж |
| Мезоморф | 4 (3 основных + 1 перекус) | Сбалансированные порции |
| Эндоморф | 3 (без перекусов) | Крупные, насыщающие порции, акцент на белок + жиры, углеводы ↓ |

**Распределение:**
- Завтрак: 25–30% калоража
- Обед: 35–40% калоража
- Ужин: 20–25% калоража
- Перекусы (если есть): 10–15% калоража

**Правила приёмов пищи:**
- Завтрак — белок + жиры + клетчатка (НЕ углеводный)
- Обед — белок + сложные углеводы + овощной салат + жиры
- Ужин — белок + овощи + жиры (минимум углеводов), за 3 часа до сна
- Перекусы — с учётом гликемического ответа и баланса микроэлементов

### ШАГ 5. Подбор рецептов

На основе выбранных продуктов и ограничений по здоровью прописать:

| Поле | Описание |
|------|----------|
| Название блюда | Понятное, аппетитное |
| Ингредиенты | С граммовками |
| Пошаговое приготовление | Краткое, но полное |
| Размер порции | В граммах |
| КБЖУ порции | Калории / Б / Ж / У |

**Правила подбора:**
- Безглютеновые крупы: гречка, рис, киноа, амарант, пшено (если нет ИР)
- Безмолочные альтернативы: кокосовое/миндальное молоко, козий сыр (если допустим)
- При запорах: акцент на сырые салаты, свёклу, чернослив, семена льна
- При диарее: акцент на варёные/тушёные блюда, овсянка, рисовый отвар
- При АПК: строго по зелёному списку из `references/10_antiparasitic.md`
- При AIP: исключить паслёновые, яйца, орехи, семена, бобовые

### ШАГ 6. Составление меню на запрашиваемое количество дней

**Формат таблицы на каждый день:**

| Приём пищи | Время | Блюдо | Краткий рецепт / состав | КБЖУ |
|---|---|---|---|---|
| Завтрак | 08:00 | ... | ... | ... |
| Перекус | 11:00 | ... | ... | ... |
| Обед | 13:00 | ... | ... | ... |
| Полдник | 16:00 | ... | ... | ... |
| Ужин | 19:00 | ... | ... | ... |

**Итого за день:** ___ ккал / Б ___ г / Ж ___ г / У ___ г

**Правила:**
- Блюдо не повторяется более 1 раза в 3 дня
- Жирная рыба — 3–5 раз в неделю
- Овощной салат — ежедневно (обед минимум)
- Вода — напоминание о суточной норме (30 мл × кг массы тела)
- Можно дополнительно отметить блюда для meal-prep (приготовить заранее / заморозить)

### ШАГ 7. Формат предоставления

- Меню в виде таблиц на каждый день
- Отдельный раздел: список покупок на неделю
- Рецепты: полные, с пошаговым приготовлением
- Отметить блюда, которые можно заранее приготовить/заморозить

### ШАГ 8. Корректировки

Через 5 дней (или по запросу клиента) провести оценку:

| Параметр | Оценка |
|----------|--------|
| Самочувствие | Лучше / без изменений / хуже |
| Стул | Нормализовался / без изменений / ухудшился |
| Вздутие | Есть / нет |
| Энергия | Повысилась / без изменений / снизилась |
| Голод / насыщение | Достаточно / голодно / переедание |
| Вкус / удобство | Нравится / надо менять рецепты |

На основании оценки — внести корректировки в меню.

## Типы меню (какой выбрать)

| Тип меню | Когда назначать | Ключевые правила |
|---|---|---|
| Безмолочное + безглютеновое | По умолчанию для большинства клиентов | Исключить глютен и молочку, тест на переносимость через 3 мес. |
| AIP (аутоиммунный протокол) | При АИЗ (АИТ, ревматоидный артрит, псориаз) | Исключить: паслёновые, яйца, орехи, семена, бобовые, молочку, глютен |
| Лакто-вегетарианское | По запросу клиента-вегетарианца | Бобово-злаковые комбинации для аминокислот, кунжут для Са |
| 1300 ккал | Для похудения (эндоморф, ИР) | Три приёма, без перекусов, КБЖУ по каждому приёму |
| При запорах | СИБР, слабая перистальтика | Сырые салаты, свёкла, квиноа, чернослив |
| При диарее | Воспаление кишечника | Варёные блюда, овсянка, рис, кисель |
| Для диабетиков / ИР | ИР, диабет II типа | 3 приёма, 10–12 ч ночной перерыв, исключить высокий ГИ |
| Антипаразитарное | Во время АПК | Строго по зелёному/красному спискам из antiparasitic |
