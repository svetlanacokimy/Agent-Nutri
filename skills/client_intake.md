# Скилл: Первичный приём клиента

## Глобальные правила вывода (ОБЯЗАТЕЛЬНО)

1. Финальные документы (profile, анализ, рекомендации, client_report и т.д.) оформляются БЕЗ markdown-таблиц. Формат только структурированный текст.
2. В режиме EXPERT использовать живой академический стиль: объяснять клиническую логику чек-листа и сбор данных в привязке к рискам клиента.
3. При первом упоминании любой аббревиатуры давать полную расшифровку в контексте предложения. Перед выводом — обязательная самопроверка расшифровок.

Примечание: внутренние служебные таблицы допустимы только как черновая логика сбора данных; в итоговый документ они НЕ попадают.

## Когда активируется

- Новый клиент обращается впервые
- Запрос на «что сдать перед консультацией»
- Создание карточки/профиля клиента

## Алгоритм работы

### Шаг 1. Сбор данных по анкете

Запросить у клиента (или взять из предоставленного файла) ответы на 19 вопросов:

| # | Вопрос | Зачем нужен |
|---|--------|-------------|
| 1 | Фамилия и имя | Идентификация, создание папки |
| 2 | Возраст | Влияет на референсы, КБЖУ |
| 3 | Город проживания | Йоддефицитные регионы, экология |
| 4 | Рост и вес | Расчёт ИМТ, КБЖУ, дозировок |
| 5 | Желаемый вес | Определение цели (похудение/набор/поддержание) |
| 6 | Перенесённые/хронические заболевания, операции | Анамнез, ограничения |
| 7 | Принимаемые лекарства на постоянной основе | Взаимодействия, побочки |
| 8 | Количество детей | Репродуктивный анамнез |
| 9 | Семейное положение | Контекст образа жизни |
| 10 | Продукты, которые не едите никогда | Для меню |
| 11 | Изжога, вздутия, брожение, метеоризм — после каких продуктов, когда | ЖКТ-проблемы |
| 12 | Усталость после обеда, хочется ли полежать | Признак ИР, гипоацидности |
| 13 | Что хотите улучшить/изменить/убрать в питании | Мотивация, цели |
| 14 | Опыт веганства/вегетарианства (сколько, когда) | Дефициты В12, железа, цинка |
| 15 | Курение, алкоголь | Факторы риска, дефицит С, В-группы |
| 16 | Стрессовый характер жизни | Кортизол, надпочечники, магний |
| 17 | Количество воды в день, следите ли | Гидратация, ЖКТ |
| 18 | Любите ли сырую еду (фрукты, салаты) | КЩБ, ферментативная активность |
| 19 | Какой вопрос хотите решить на консультации | Главный запрос |

### Шаг 2. Создание профиля клиента

Создать файл `clients/[ФИО]/profile.md` со структурой:

```markdown
# Клиент: [ФИО]

## Основные данные
- Возраст: 
- Город: 
- Рост: ___ см | Вес: ___ кг | ИМТ: ___
- Желаемый вес: 

## Анамнез
- Хронические заболевания: 
- Операции: 
- Принимаемые лекарства: 
- Дети: 
- Курение: | Алкоголь: 

## Жалобы и симптомы
- Главный запрос: 
- ЖКТ (изжога/вздутие/метеоризм): 
- Усталость после обеда: 
- Стрессовый фон: 

## Питание
- Не ест: 
- Опыт вегетарианства: 
- Вода в день: 
- Сырая еда: 

## Цели
- [ ] ...
```

### Шаг 3. Формирование чек-листа анализов

На основании жалоб и анамнеза сформировать индивидуальный список анализов.

#### Базовый чек-лист (для всех клиентов)

| Блок | Анализы |
|------|---------|
| **ОАК** | Общий анализ крови с лейкоцитарной формулой + ретикулоциты |
| **Обмен железа** | Ферритин, железо сывороточное, трансферрин |
| **Биохимия** | Общий белок, альбумин, билирубин (общий/прямой/непрямой), АЛТ, АСТ, ГГТ, ЩФ, мочевина, креатинин, мочевая кислота |
| **Углеводный обмен** | Глюкоза, инсулин, гликированный гемоглобин |
| **Липидный профиль** | Общий холестерин, ЛПВП, ЛПНП, триглицериды |
| **Гомоцистеин** | Гомоцистеин |
| **Витамины** | Витамин D (25-OH), В12, В9 (фолат), В6 |
| **Минералы** | Магний, цинк, медь, селен, кальций |
| **ЩЖ** | ТТГ, Т3 свободный, Т4 свободный, АТ-ТПО, АТ-ТГ |
| **Воспаление** | СРБ (ультрачувствительный) |

#### Расширенный список (по показаниям)

| Показание | Дополнительные анализы |
|-----------|----------------------|
| Подозрение на ИР | С-пептид, HOMA-IR |
| Проблемы с ЖКТ | Копрограмма, кальпротектин |
| Выпадение волос | ДГЭА-С, тестостерон своб., пролактин |
| Гинекологические жалобы | Эстрадиол, прогестерон (на 21–22 д.ц.), ЛГ, ФСГ |
| Подозрение на паразитоз | Антитела к аскаридам, лямблиям, описторхозу, токсокарам |
| Отёки, почки | Общий анализ мочи, проба Реберга |
| Суставы, кости | Кальцитонин, паратгормон, остеокальцин |
| Подозрение на АИТ/ВЭБ | Антитела к ВЭБ (IgG VCA, IgG EBNA, IgM VCA), АТ-ТГ |
| Йоддефицит | Йод в моче / сыворотке |

### Шаг 4. Инструкция для клиента

Предоставить клиенту:
1. **Чек-лист анализов** — с указанием, что сдавать натощак, условия подготовки
2. **Дневник питания** — вести 3 дня, ничего не меняя в привычном рационе
3. **Рекомендация лаборатории** — LAB4U, Гемотест, Инвитро (по городу)
4. **Договориться о дате сдачи** — контролировать, чтобы клиент сдал
5. **После сдачи** — назначить дату платной консультации

### Шаг 5. Создание структуры папки клиента

```
clients/
└── [ФИО]/
    ├── profile.md          ← анкета + анамнез
    ├── blood_tests.md      ← таблица анализов (после сдачи)
    ├── recommendations.md  ← рекомендации (после консультации)
    └── diary/              ← дневник питания, прогресс
```

## Этапы работы с клиентом (памятка)

| Этап | Действие |
|------|----------|
| 1. Подготовка (заочно) | Клиент: дневник питания 3 дня + анкета + сдача анализов |
| 2. Консультация | Разбор анализов → рекомендации → нутрицевтики → меню (опционально) |
| 3. Реализация | Клиент следует рекомендациям 1–3 месяца |
| 4. Контроль | Пересдача анализов → сравнение динамики → корректировка |
